Tentang Saya

Perkenalkan nama saya Putri Difa'anis Sakinah Hijriyyah,biasanya teman-teman saya memanggil saya 'Anis'.Saya lahir di Tangerang pada 5 Maret 2004. Saya anak ke-5 dari 6 bersaudara,saya pindah ke Tegal saat umur saya 4 tahun akhirnya pendidikan saya dari TK sampai sekarang hanya didaerah Tegal saja.Riwayat pendidikan saya TK di Aisiyah Slawi,lalu SD Muhammadiyah Slawi,SMPIT Luqman Al-Hakim Slawi lanjut SMAIT Luqman Al-Hakim Slawi, Lalu sekarang saya melanjutkan pendidikan saya di Politeknik Baja Tegal dengan Prodi D3 Teknik Informatika.Mengapa saya mengambil prodi tersebut karna saya rasa saya mampu dibidang teknologi dan saya suka untuk belajar tentang teknologi.Lalu saya merasa mempunyai skill dibidang Multimedia kearah fotografi dan edit mengedit foto atau video.Sejak SMP saat organisasi mengadakan acara pasti saya selalu yang terpilih menjadi tim dokumentasi,saya berharap dengan mengambil prodi teknik informatika dan saya yang mempunyai skill dibidang Multimedia bisa jadi peluang sayang meraih kesuksesan.Nah jadi gitu temen-temen sepertinya cukup perkenalan dari saya.
